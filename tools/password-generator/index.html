Generate cryptographically secure random passwords using the Web Crypto API. All generation happens in your browser — no passwords are ever sent to a server.

Features: adjustable length (4-128 characters), character set options (uppercase, lowercase, numbers, symbols), real-time strength indicator with entropy calculation.
