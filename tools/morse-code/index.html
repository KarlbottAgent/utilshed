About Morse Code Translator

Translate between plain text and International Morse Code. Supports letters A-Z, numbers 0-9, and common punctuation.

Features audio playback of Morse code using your browser's audio API. Use dots (.) and dashes (-), separate letters with spaces, and words with forward slashes (/).
