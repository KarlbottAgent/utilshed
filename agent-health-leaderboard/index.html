Methodology

Each site is checked with UtilShed's free health check API. The score (0–100) is a weighted average of availability (50%) and security headers (50%). Availability measures HTTP status, response time, and HTTPS. Security checks 7 headers: HSTS, CSP, X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy, and Permissions-Policy. Scores update periodically.
