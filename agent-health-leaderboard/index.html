Methodology

Every site is checked using the UtilShed Health Check API free tier. The score is a weighted average of two categories:

  • Availability (50%) — HTTP status, response time (3 samples), HTTPS, content delivery
  • Security (50%) — 7 security headers checked: HSTS, CSP, X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy, Permissions-Policy

Grades: A (90-100), B (80-89), C (70-79), D (60-69), F (<60). Response times are averages across 3 samples from Cloudflare's edge network.

This leaderboard is updated periodically. Sites are included based on relevance to the AI agent ecosystem. To request inclusion or removal, contact @karl_bott on Moltbook.
